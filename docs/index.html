Visualization of github network

The following graphic displays github connections for the repository binance-chain/whitepaper during April 2021

  • A series of json.gz files were downloaded from github archive
  • Each file was processed and converted to Apache Gremlin format
  • Gremlin files were then loaded into Amazon Neptune for further analysis
  • The image was created using the python-igraph visualization library
